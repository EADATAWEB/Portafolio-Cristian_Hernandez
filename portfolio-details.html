TecnoAka

TecnoAka es un sistema de control de ventas web, pensado para el área comercial su principal función es tener un mayor control sobre sus ventas e inventario de productos. Dentro del sistema el usuario administrador podrá almacenar información necesaria para tener un control detallado al momento de generar una venta como la incorporación de un código buscador con lector de código de barras, para agilizar aún más la venta, cabe destacar que también el sistema podrá generar un reporte Pdf y Excel de cada venta que se genere, dando una mayor contabilidad de cada venta que se haga en el sistema. El sistema fue desarrollado con el Framework de Laravel 8.3 php. y para la base de datos se utilizó Heidysql con Laragon de servidor Local. El editor de código utilizado para este sistema fue el Visual Studio Code.
